## 📌 Descrizione e obbiettivi
"Il nostro progetto si impegna a fornire un sito web multifunzionale per le società sportive della provincia di Bergamo.
Il nostro obiettivo è dare a ogni organizzazione l'opportunità di farsi conoscere non solo nel mondo reale, ma anche online."

## 🛠️ Tecnologie utilizzate
Il progetto è stato sviluppato utilizzando HTML per la struttura delle pagine, CSS per lo stile e il layout grafico e JavaScript per la gestione della logica lato client, sql per la creazione del database e php per connettere client e server. È stato utilizzato GitHub per il versionamento del codice e il LocalStorage del browser per la gestione delle preferenze relative ai cookie.

## 🌐 Struttura del progetto
Il repository è organizzato in modo semplice e modulare. Il file principale è index.html, da cui parte l’intero sito. Le risorse grafiche e di stile sono contenute nella cartella /css, gli script JavaScript nella cartella /js e la documentazione relativa alla cookie policy nella cartella /docs, in particolare nel file cookie-policy.html.

## 🔐 Gestione Cookie e GDPR
Il progetto include un sistema di gestione dei cookie progettato per rispettare i principi del GDPR (Regolamento UE 2016/679). Al primo accesso al sito viene mostrato un banner che informa l’utente sull’utilizzo dei cookie e consente di scegliere tra diverse opzioni: accettare tutti i cookie, rifiutare quelli non necessari oppure personalizzare le preferenze. Le scelte effettuate vengono salvate nel browser tramite LocalStorage, permettendo al sito di ricordare le preferenze dell’utente nelle visite successive.

Un aspetto fondamentale è che eventuali script di tracciamento o servizi esterni vengono attivati solo dopo aver ottenuto il consenso esplicito dell’utente, in linea con le normative europee. È inoltre presente una pagina dedicata alla Cookie Policy, accessibile dal sito, che descrive nel dettaglio le tipologie di cookie utilizzati e le modalità di gestione del consenso.

## 🚀 Come avviare il progetto
Per eseguire il progetto è sufficiente clonare il repository da GitHub utilizzando il comando git clone seguito dall’URL del repository. Una volta scaricati i file, è possibile aprire il progetto semplicemente avviando il file index.html con un qualsiasi browser web moderno, senza necessità di server o configurazioni aggiuntive.
